Why Python Learning Lab?
  • Learn a real skill. Python is one of the most popular programming languages in the world — used by professionals, scientists, and creators everywhere.
  • Taught by a kid, for kids. This course is designed with young learners in mind, so the pace, language, and examples actually make sense to you.

About the Host

Eshaan Buddhisagar is a young podcaster, author, and programmer with years of experience in Python. He has published Python Learning Lab (Book), a book to teach kids Python easily. We'll cover the same topics from the book, so you can follow along if you have it.

What You'll Learn

This is the plan for every single week in the course.

Session 1 · June 14
Getting Started

Install Python, write your first programs, learn print, basic math & variables.

Session 2 · June 21
Variables & Data Types

Understand strings, numbers, booleans, and practice with small exercises.

Session 3 · June 28
Operators & Conditionals

Learn how Python does math and logic — and learn about if conditions.

Session 4 · July 5
Loops

Repeat actions with for & while loops, explore range(), and solve loop puzzles.

Session 5 · July 12
Lists & Tuples

Store and work with collections of data. Iterate and transform them.

Session 6 · July 19
Dictionaries

Learn key-value storage and sets — then apply them by making your own cipher.

Session 7 · July 26
Functions

Write reusable code with parameters, return values, and plenty of practice.

Session 8 · Aug 2
Classes and Objects

Learn about object-oriented programming and how to use classes/objects.

Session 9 · Aug 9
Mini Project — Part 1

Plan and start coding your own Python project from scratch.

Session 10 · Aug 16
Mini Project — Part 2

Finish your project, present it, and celebrate what you've built!
